## Scripts

- **`lint`**: Lint and auto-fix code using `ts-standard`.  
  ```bash
  npm run lint
  ```
- **`build`**: Compile TypeScript to JavaScript.  
  ```bash
  npm run build
  ```
- **`start:dev`**: Run the app in development mode with `ts-node`.  
  ```bash
  npm run start:dev
  ```
- **`start:prod`**: Run the compiled app in production mode.  
  ```bash
  npm run start:prod
  ```
